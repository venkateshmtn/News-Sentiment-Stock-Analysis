\# 📊 News Sentiment Analysis with Stock Correlation



\## 🚀 Overview



This project analyzes financial news sentiment and correlates it with stock price movements using NLP and time-series analysis.



\---



\## 🧠 Features



\* News data collection using API

\* Text cleaning and preprocessing

\* Sentiment analysis using NLTK (VADER)

\* Time-series aggregation

\* Stock data integration using yfinance

\* Interactive dashboard using Streamlit



\---



\## 🛠️ Tech Stack



\* Python

\* Pandas, NumPy

\* NLTK

\* Seaborn, Matplotlib

\* yfinance

\* Streamlit



\---



\## 📁 Project Structure

news-sentiment-analysis/

\## ▶️ How to Run



\### 1. Install dependencies



```

pip install -r requirements.txt

```



\### 2. Run dashboard



```

streamlit run app.py

```

\* Sentiment distribution
